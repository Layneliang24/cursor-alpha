# 博客模块 - 原型图（摘要）

## 列表页（桌面）
```
┌──────────────────────────────────────────────────────────┐
│ 搜索 [关键字] (分类▼) (标签▼)           [+ 新建文章]       │
├──────────────────────────────────────────────────────────┤
│ 标题                     │ 分类 │ 标签        │ 状态 │ 时间 │
├──────────────────────────────────────────────────────────┤
│ 深入理解Vue3             │ 前端 │ vue, ui     │ 发布 │ 今天 │
│ Django REST最佳实践      │ 后端 │ django, api │ 草稿 │ 昨天 │
└──────────────────────────────────────────────────────────┘
```

## 详情页（桌面）
```
┌───────────────────────────────────────────┐  ┌──────────────┐
│ 标题                                     │  │ 目录/信息    │
│ 作者 | 时间 | 分类 | 标签                 │  │ 相关文章      │
├───────────────────────────────────────────┤  └──────────────┘
│ Markdown 渲染区（代码高亮/图片/公式）     │
└───────────────────────────────────────────┘
```

```
│ 爬虫策略管理                          [ 新建策略 ]       │
├──────────────────────────────────────────────────────────┤

```
